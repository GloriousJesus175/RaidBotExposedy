# Exposed Society

Projeto desenvolvido pela equipe Exposed Dev's Team.

## Descrição

Aplicação desenvolvida em .NET 8.0 para Windows.

## Requisitos

- .NET 8.0 SDK ou superior
- Windows (x64)

## Build

Para compilar o projeto:

```bash
dotnet build -c Release
```

## Publicação

Para criar um executável Windows:

```bash
publish.bat
```

O executável será gerado na pasta `publish/` com o ícone `Exposed.ico` embutido.
